## 简介<a name="section11660541593"></a>

通话设置应用是OpenHarmony标准系统中预置的系统应用，为用户提供通话设置、移动数据设置、SIM卡管理设置、卫星通信设置等功能。

## 目录<a name="section161941989596"></a>

```
callsetting
├─ product
│  ├─ default
│  │  └─ src
│  │     └─ main
│  │        ├─ ets
│  │           ├─ abilities        # 应用能力模块
│  │           ├─ application      # 应用程序相关代码
│  │           ├─ base             # 基础功能
│  │           ├─ common           # 通用功能
│  │           ├─ databases        # 数据库相关
│  │           ├─ pages            # 页面组件
│  │           ├─ taskpool         # 任务池相关
│  │        ├─ resources           # 资源文件存放目录
│  └─ wearable                     # 穿戴设备定制
│     └─ src
│        └─ main
│           ├─ ets                 # ets代码目录
│           └─ module.json5        # 模块配置文件
├─ common                      # 通用逻辑存放目录
├─ feature                     # 相关模块业务逻辑存放目录
├─ LICENSE                     # 许可文件
├─ signature                   # 证书文件目录
```
