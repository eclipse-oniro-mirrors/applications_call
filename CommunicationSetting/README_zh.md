## 简介<a name="section11660541593"></a>

通信设置应用是OpenHarmony标准系统中预置的系统应用，为用户提供流量统计、SIM卡管理、应用联网权限控制等功能。

## 目录<a name="section161941989596"></a>

```
communicationsetting
├─ entry
│  └─ src
│     └─ main
│        ├─ ets
│           ├─ application        # 应用程序相关代码
│           ├─ constants          # 应用常量对象目录
│           ├─ Controller         # 控制器层代码目录
│           ├─ data               # 数据层代码目录
│           ├─ entryability       # Ability与ExtentionAbility存放目录
│           ├─ feature            # 图表功能模块
│           ├─ model              # 模型层代码目录
│           ├─ pages              # 页面目录
│           ├─ pagesComponents    # 页面组件
│           ├─ presenter          # 展示器层代码目录
│           ├─ service            # 服务层代码目录
│           ├─ util               # 工具类目录
│           ├─ view               # 视图层代码目录
│           ├─ viewmodel          # 视图模型层代码目录
│           └─ workers            # 工作线程目录
│        ├─ resources             # 资源文件存放目录
├─ product                        # 产品定制目录
│  └─ wearable                    # 穿戴设备定制
│     └─ src
│        └─ main
│           ├─ ets                # ets代码目录
│           └─ resources          # 资源文件存放目录
├─ LICENSE                        # 许可文件
├─ signature                      # 证书文件目录
```
